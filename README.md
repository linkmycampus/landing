# LinkMyCampus Landing Page

Marketing site for **LinkMyCampus** — campus social for university students (visible + anonymous profiles, feed, groups, reels).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Marketing homepage |
| `/waitlist` | Student waitlist form |
| `/ambassador` | Campus ambassador application |

## Brand

- **Fonts:** Poppins (headings), Nunito (body)
- **CTA buttons:** `#1a2b56`
- **Logo:** `src/assets/logo.png`

## Development

```bash
npm install
npm run dev
```
